# Week 7 V3 Results

The normalized-gradient rollback Colab run writes its outputs here. This folder
is independent from the completed v1, v2, and trial-8 audit directories.

Expected generated content includes:

- per-trial guard decisions and selection predictions;
- gradient norms, conflict rates, projection rates, and normalization factors;
- resumable controller metadata;
- candidate-best and selected adapters;
- full candidate evaluations and cross-week comparisons;
- `metrics.json` and `week7_v3_normalized_report.md`.

Rolling accepted and best adapters are also backed up under the separate
`week7-v3-normalized-resume-state` GitHub Release tag.
